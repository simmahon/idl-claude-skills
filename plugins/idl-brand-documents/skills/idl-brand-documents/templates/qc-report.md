# Template — QC Inspection Report

**Cover label:** `QC INSPECTION REPORT`
**Cover title:** product name
**Cover meta items:** Factory · Inspection date · Result (PASS / FAIL / CONDITIONAL)

## Page 2 — Inspection Summary
- 3-card grid: Factory · Date · Result
- Section: Test Results — `data-table` (Test, Standard, Result, Notes)

## Page 3 — Defects + Action
- Section: Defects Identified — `data-table` (Defect, Severity, Quantity, Photo ref)
- Section: Overall Result — `info-box` (vermillion border ONLY if FAIL or CONDITIONAL; blue border if PASS)
- Section: Required Corrective Actions — `bullet-list`

## Notes
- If only a single defect is reported, still use a data-table (don't collapse to text).
- Severity values use the labels: Critical, Major, Minor.
- If the factory report is in another language, translate to UK English. Flag terms that are ambiguous.
