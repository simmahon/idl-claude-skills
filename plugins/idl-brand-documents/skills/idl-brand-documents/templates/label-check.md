# Template — Label Compliance Check

**Cover label:** `LABEL COMPLIANCE CHECK`
**Cover title:** product name + market (e.g. "Tydi Hangers — UK & EU")
**Cover meta items:** Market(s) · Source (text input or photo) · Date checked

## Content page(s)
- Section: Product Details — `card-grid` (3 cards: name, category, claimed material/spec)
- Section: Required Label Fields — `data-table` (Field, Requirement, Present / Missing / Incorrect)
- Section: Assessment Summary — `info-box` (vermillion border if any "Missing" or "Incorrect"; blue otherwise)
- Section: Recommendations — `bullet-list` (specific changes to the label, regulation references where applicable)

## Notes
- Status values use exactly: Present · Missing · Incorrect · Verify
- Cite regulation numbers where the requirement comes from a specific reg (e.g. "GPSR Article 9").
- If input was a photo and any text is illegible, list those fields under "Verify" not "Missing".
