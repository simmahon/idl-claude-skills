# Template — Compliance Document Register

**Cover label:** `COMPLIANCE DOCUMENT REGISTER`
**Cover title:** brand or product range (e.g. "Pest X Pro — UK Range")
**Cover meta items:** Scope · Last reviewed · Owner

## Content
- Single full-width `data-table` with these columns: Document Type · Required For · Status · Expiry Date · Owner · Notes
- Use `.cat` rows (light-grey full-width header rows) to group entries by brand or product category.
- If the table runs past one page, split into additional `.a4` content pages keeping the same header.

## Notes
- Status values: Current · Expiring soon (within 90 days) · Expired · Missing
- Expiry date format: `YYYY-MM-DD`.
- For documents with no expiry, write `n/a` (not blank).
- Sort within each `.cat` group: Missing first, then Expired, then Expiring soon, then Current.
