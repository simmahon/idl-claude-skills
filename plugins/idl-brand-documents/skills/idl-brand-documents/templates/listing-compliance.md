# Template — Amazon Listing Compliance Check

**Cover label:** `LISTING COMPLIANCE CHECK`
**Cover title:** product name + marketplace (e.g. "Tydi Hangers — Amazon UK")
**Cover meta items:** ASIN · Marketplace · Category

## Content
- Section: Listing Details — `card-grid` (3 cards: ASIN, marketplace, category)
- Section: Claims Assessment — `data-table` (Claim text, Policy reference, Pass / Fail / Flag)
- Section: Restricted Terms Found — `info-box` with vermillion border for any Fail; blue-bordered "no restricted terms identified" box if clean
- Section: Recommended Copy Changes — `two-col-grid` (each pair: Current vs Recommended)
- Section: Summary Verdict — `info-box` (Approved · Needs amendments · Suspend pending review)

## Notes
- Status values in Claims: Pass · Fail · Flag (Flag = ambiguous, needs human review)
- Cite Amazon policy by name (e.g. "Restricted Products — Pesticides", "Drug Claims policy").
- Recommended copy must be drop-in usable, not a description of what to change.
- Pest control and supplements: extra scrutiny — see references/company-context.md if needed.
