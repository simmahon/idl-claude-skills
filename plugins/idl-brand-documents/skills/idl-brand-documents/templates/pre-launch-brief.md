# Template — Pre-Launch Compliance Brief

**Cover label:** `PRE-LAUNCH COMPLIANCE BRIEF`
**Cover title:** product name
**Cover meta items:** Brand · Target market(s) · Planned launch date

## Page 2 — Overview & Regulations
- Section: Product Overview — `card-grid` (3 cards: category, materials, intended use)
- Section: Applicable Regulations — `data-table` (Regulation, Market, Status)

## Page 3 — Documentation & Labels
- Section: Required Documentation — `data-table` (Document, Required for, Owner, Status)
- Section: Label Requirements — `two-col-grid` (each card = required field vs status)
- Section: Amazon Category Restrictions — `info-box` for any restrictions or "no restrictions identified" if clean

## Final page — Priority Actions
- Section: Priority Actions — `info-box` with vermillion border + `bullet-list` of pre-launch must-dos

## Notes
- Regulation "Status" values: Met · In progress · Not started · Verify current guidance
- If a regulation has changed recently, flag with "Verify current guidance" rather than asserting.
- Priority Actions list must be specific and dated where possible.
