# Template — QC Trend Summary

**Cover label:** `QC TREND SUMMARY`
**Cover title:** period covered (e.g. "Q2 2026" or "April 2026")
**Cover meta items:** Period · Brands covered · Number of inspections

## Content
- Section: Products Reviewed — `data-table` (Product, Brand, Factory, Inspections, Pass rate)
- Section: Key Findings — 2–4 stacked `info-box` callouts (one per major finding)
- Section: Recurring Issues — `data-table` (Issue, Frequency, Affected products, Severity)
- Section: Factory Performance Summary — `two-col-grid` (one card per factory: pass rate, recurring issue, recommendation)
- Section: Recommended Actions — `bullet-list` (prioritised, with owner if known)

## Notes
- Pass rate format: percentage with raw count in parens, e.g. `92% (23 of 25)`.
- Frequency in Recurring Issues: integer count over the period.
- Order Key Findings most-impactful first.
